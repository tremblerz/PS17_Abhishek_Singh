###Running this project
There are some software requirements to be considered in order to run this program.
1. django >= 1.9
2. rabbit-mq server
3. Mozilla Firefox with its geckodriver
4. Celery
5. Some python modules as listed -
	a) Scrapy
	b) Celery
	c) selenium
	d) re
	e) random
	f) chartit
